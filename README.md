# Webscraping-Project

## Using this code

Make a virtual environment:
On Mac:

```
$ python3 -m venv venv
$ source venv/bin/activate
```

On Windows:
```
$ python3 -m venv venv
$ venv/scripts/activate.ps1
```

And make sure all packages are up to date:

```
(venv) $ pip3 install -r requirements.txt
```
